I. 分类指标

Confusion Matrix

  • TP：预测正确，真实类别与模型预测都为Positive
  • FP：预测错误，模型预测为Positive，但真实类别为Negtive
  • FN：预测错误，模型预测为Negtive，但真实类别为Positive
  • TN：预测正确，真实类别和模型预测都为Negtive

Accuracy

Precision

  • 预测为Positive的样本中，有多大比例是真的Positive

Recall

  • 所有真实类别为Positive的样本中，能找出多少

PR曲线

  • 一般来说，Recall越高则Precision越低
  • 根据两者值可绘制PR曲线，曲线越靠近右上角说明性能越好
  • 缺点：当正负样本分布发生变化，其形状会发生大幅改变

F1-score

  • 同时关注正样本的Precision与Recall，且不使用Acc
  • 只有当Precision和Recall都很高时，F1才会高

ROC

  • 上述指标都是单一的数值指标
  • ROC可观察算法在不同参数下的表现，并且对正负样本分布不敏感(正负样本比例发生变化，ROC曲线也能基本保持不变)
  • ROC曲线中，横坐标为FPR，纵坐标为Recall

  • 下图为一个ROC曲线，曲线接近左上角时表示性能好
    • 点(0,0)：分类器预测所有样本都是负样本；
    • 点(1,1)：分类器预测所有样本都是正样本；
    • 点(0,1)：最优分类器，所有样本都被正确分类；
    • 点(1,0)：最差分类器，所有样本都被错误分类；

  • 绘制方法：
    • 截断点法：例如在二分类中，模型的输出一般为”样本为P(正)的概率”，因此可以先调节一个阈值(当模型的输出 > 阈值 => 样本为正)；然后计算不同阈值下的TPR与FPR，就得到了图像上对应的点；依次连接各点即可
    • 直接法：
      1. 根据样本标签统计正负样本的数量分别为P，N
      2. 将横轴刻度间隔设置为 1/N，纵轴间隔设置为 1/P
      3. 根据模型输出的预测概率对样本从高到低排序
      4. 按这个顺序依次遍历样本，从零点开始绘制ROC曲线：每遇到一个正样本，沿纵轴绘制一个刻度；每遇到一个负样本，沿横轴绘制一个刻度
      5. 直到遍历完所有样本，曲线停在(1,1)，结束

AUC

  • AUC是ROC曲线下的面积，可直接量化评估分类器的性能，AUC越大说明模型越可能把真正的正样本预测为“正样本”，分类性能越好
  • AUC表示一个概率：随机挑选一个正样本与一个负样本，分类器对正样本的预测值(输出的概率)高于对负样本的预测值的概率
  • 性能表现：
    • AUC=1：理想的分类器，能识别所有类别；
    • AUC=0.5：相当于随机预测，这种分类器不可用；
    • AUC>0.5：优于随机预测，是实际情况下大部分分类器的情况；
    • AUC<0.5：不如随机预测

II. 回归指标

MSE/RMSE

MAPE

  • 平均绝对百分比误差

R2

  • 也叫拟合优度，决定系数
